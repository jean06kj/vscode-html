FAQ - Perguntas Frequentes

O que é HTML?

HTML (HyperText Markup Language) é a linguagem de marcação usada para criar páginas na web. Ele estrutura o conteúdo através de elementos como títulos, parágrafos, links e imagens.

Para que serve a tag <div>?

A tag <div> é usada para agrupar elementos HTML, permitindo aplicar estilos ou manipulações em conjunto com CSS ou JavaScript.

Qual a diferença entre <span> e <div>?

A tag <span> é usada para estilizar partes pequenas do texto, sendo um elemento inline. Já a <div> é um elemento de bloco, utilizado para agrupar conteúdos maiores.
