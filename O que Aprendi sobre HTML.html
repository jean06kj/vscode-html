O que Aprendi sobre HTML

Introdução ao HTML

O HTML (HyperText Markup Language) é a linguagem de marcação básica usada para criar páginas na web.
Até agora, aprendi sobre a estrutura fundamental de um documento HTML, como os elementos principais como <html>, <head>, <body>, e outros.

Estrutura de um Documento HTML

Um documento HTML é estruturado em uma hierarquia de elementos. O <html> envolve todo o conteúdo da página. Dentro de <head>, definimos as metainformações, como o título da página e a codificação de caracteres.

A tag <body> contém o conteúdo visível da página, como textos, imagens e links.

Formatação de Texto

No HTML, podemos formatar texto usando várias tags. Por exemplo:

  • strong e em para destacar o texto.
  • del para riscar texto, indicando algo removido.
  • ins para sublinhar e mostrar inserções ou adições.

Aprendi a utilizar essas tags para melhorar a apresentação do conteúdo, tornando-o mais legível e interessante.

Entidades HTML

As entidades HTML são usadas para representar caracteres especiais. Por exemplo:

  • < para representar o símbolo menor que (<).
  • © para representar o símbolo de copyright (©).

Essas entidades são essenciais para evitar conflitos com as tags HTML e garantir que caracteres especiais sejam exibidos corretamente na página.

Exibindo um Trecho de Texto Formatado

    Exemplo de código HTML formatado:
    <html>
        <head><title>Página de Exemplo</title></head>
        <body>
            <h1>Bem-vindo ao meu site</h1>
            <p>Esta é uma página de exemplo</p>
        </body>
    </html>

Conclusão

Em resumo, o HTML é a espinha dorsal de qualquer página web.
Ele define a estrutura, a formatação e a apresentação dos conteúdos, proporcionando uma base para o desenvolvimento de websites completos.
Com o aprendizado das tags e suas funcionalidades, posso agora criar páginas bem estruturadas e com formatação adequada.
